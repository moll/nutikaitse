Nutikaitse
==========

Installimine
------------
```sh
bundle install
```


Käivitamine
-----------
Dokumendi genereerimiseks ja brauseris vaatamiseks käivita nii lokaalne server
kui ka kompileerimisprotsess:

```sh
make -j2 server autocompile
```

Dokumendi nägemiseks ava brauseris <http://localhost:4000>.
